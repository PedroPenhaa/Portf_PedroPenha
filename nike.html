Site Nike Collection

Sobre

O site Nike Collection é um projeto cuidadosamente concebido para aprimorar minhas competências em HTML, CSS e JavaScript. Este empreendimento oferece uma simulação de uma loja de calçados Nike, abrangendo a criação de páginas de produtos, um carrinho de compras interativo e uma página de finalização de compra, enquanto consolida os princípios fundamentais do desenvolvimento web. Esta adição representa uma contribuição prática e enriquecedora ao meu portfólio, evidenciando meu compromisso com a excelência no desenvolvimento front-end.
