Projeto CDPC

Aplicativo Centro de Divulgação e Popularização da Ciência

  • Java
  • Firebase
  • Android

Aplicação mobile que atende a demanda do Centro de Divulgação e Popularização da Ciência (CDPC), do Instituto Federal, Campus Poços de Caldas. O projeto teve como intuito disponibilizar Kits portáteis de experimentos na área da ciências da natureza, que são reservador por um aplicativo para professores da rede pública de ensino de Poços de Caldas.

Restaurante Sabor Brasileiro

  • HTML5
  • CSS3
  • JS

Projeto de conclusão do curso FIC de Desenvolvimento Web, realizado no campus de Poços de Caldas. O projeto consistiu no desenvolvimento de um site completo para um restaurante fictício, o "Sabor Brasileiro." Este site foi cuidadosamente concebido para proporcionar uma experiência excepcional aos visitantes, com foco na usabilidade, design atraente e interatividade
